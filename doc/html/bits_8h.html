typedef struct _bit_chain Bit_Chain

The position of bits within bytes is numerically ordered as depicted below:

position: 01234567 01234567 01234567 ... bits: 76543210 76543210 76543210 ... / / / byte 1 byte 2 byte 3 ... Structure for DWG-files raw data storage
